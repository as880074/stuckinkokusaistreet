## 🔧 自訂調整

### 修改配色
編輯 `css/variables.css`，所有配色都定義在這裡。

### 修改 Hero 背景
編輯對應的 CSS 檔案：
- 首頁：`css/home.css`
- 行程：`css/itinerary.css`
- Day 頁：`css/day.css`（最下方有各 Day 的專屬樣式）

### 修改動畫
所有動畫使用 `@keyframes`，可在 CSS 中直接調整：
- 速度：修改 `animation` 的時間參數
- 幅度：修改 `transform` 的數值
- 關閉：移除 `animation` 屬性
